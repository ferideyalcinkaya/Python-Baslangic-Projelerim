# Python-Baslangic-Projelerim
İlk başlangıç projesi olarak "Manav Stok Yönetimi" yapacağız.
Bu proje, Python'da "matris yapısı, iç içe döngüler ve fonksiyonlar" konusunu pekiştirmek için geliştirilmiştir.
Özellikler
- Matris içinde veri arama.
- Dinamik stok güncelleme.
- Hatalı giriş ve yetersiz stok kontrolü.
